# 機能ガイド

pioyan-chat が提供する機能と使い方の一覧です。

## 1. ユーザー認証

### サインアップ

1. `http://localhost:3000/signup` にアクセス
2. メールアドレス・パスワード（8 文字以上）を入力
3. ユーザー名は省略可能（省略時はメールアドレスが表示名になります）
4. **サインアップ** ボタンをクリック

サインアップ後、JWT トークンが発行され自動的にチャット画面に遷移します。

### ログイン

1. `http://localhost:3000/login` にアクセス
2. メールアドレスとパスワードを入力
3. **ログイン** ボタンをクリック

### ログアウト

1. サイドバー左下の **⚙ 設定アイコン** をクリック
2. プロフィール設定モーダルを開く
3. **ログアウト** ボタンをクリック

ログアウト後、ログインページに戻ります。

---

## 2. プロフィール設定

サイドバー左下の **⚙ アイコン** からプロフィールを編集できます。

### 表示名の変更

1. **⚙ アイコン** → プロフィール設定モーダルを開く
2. **表示名** フィールドを編集（2〜64 文字）
3. **保存** ボタンをクリック

### アバター画像の設定

1. プロフィール設定モーダルを開く
2. アバター画像の **📤 アップロードアイコン** または **「画像を変更」** をクリック
3. JPEG / PNG / GIF / WebP 形式の画像を選択
4. **保存** ボタンをクリック

---

## 3. チャンネル

### チャンネルの種類

| 種類 | アイコン | 説明 |
|------|---------|------|
| パブリック | `#` | すべてのユーザーが参加・閲覧可能 |
| プライベート | `🔒` | メンバーのみ参加・閲覧可能 |
| DM（ダイレクトメッセージ） | `○` | 1 対 1 の会話 |

### チャンネルの作成

1. サイドバーの **チャンネル** セクション横の **＋ アイコン** をクリック
2. チャンネル名を入力（1〜80 文字）
3. 説明文を入力（任意）
4. プライベートにする場合はトグルをオン
5. **作成** ボタンをクリック

### チャンネルへの参加・切り替え

サイドバーのチャンネル名をクリックすると、そのチャンネルのメッセージが表示されます。

### チャンネルへのメンバー追加

API 経由で操作します（UI 未実装）:

```bash
curl -X POST http://localhost:8000/api/channels/{channel_id}/members \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{"user_id": "target_user_id"}'
```

## 4. メッセージ送受信

### テキストメッセージの送信

1. 中央下部の入力フィールドにテキストを入力
2. **Enter** キーまたは **→ 送信ボタン** をクリック

改行したい場合は **Shift + Enter** を使用します。

### Markdown 記法

メッセージ本文に Markdown 記法が使えます。

| 記法 | 表示 |
|------|------|
| `**太字**` | **太字** |
| `*斜体*` | *斜体* |
| `` `コード` `` | `コード` |
| ` ```コードブロック``` ` | コードブロック |
| `# 見出し` | 見出し |

### メッセージの削除

API 経由で操作します（UI 未実装）:

```bash
curl -X DELETE http://localhost:8000/api/messages/{message_id} \
  -H "Authorization: Bearer {token}"
```

## 5. リアルタイム通信

チャンネルを開くと Socket.IO によるリアルタイム通信が開始されます。

- **新着メッセージ**: 他のユーザーが送信したメッセージが即座に表示されます
- **オンライン状態**: ユーザーの接続・切断が `user_online` / `user_offline` イベントで通知されます
- **タイピングインジケーター**: メッセージを入力中の状態が他のユーザーに通知されます

---

## 6. スレッド（返信）

特定のメッセージに対してスレッド形式で返信できます。

### スレッドを開く

1. メッセージにホバーすると右上に **返信** ボタンが表示されます
2. **返信** ボタンをクリックするか、メッセージ下部の **「N replies」** リンクをクリック
3. 右サイドパネルにスレッドが表示されます

### スレッドに返信する

1. スレッドパネルの入力フィールドに返信を入力
2. **Enter** または **→ 送信** をクリック

### スレッドを閉じる

スレッドパネル右上の **✕** をクリックします。

---

## 7. ダイレクトメッセージ（DM）

### DM の開始

API 経由で操作します（UI 未実装）:

```bash
curl -X POST http://localhost:8000/api/dm \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{"user_id": "target_user_id"}'
```

DM が作成されると、サイドバーの **ダイレクトメッセージ** セクションに表示されます。

### DM の利用

サイドバーの DM リストから相手のユーザー名をクリックすると、1 対 1 の会話画面が開きます。

---

## 8. ファイルアップロード

メッセージと一緒にファイルを送信できます。

### ファイルを添付する

1. 入力フィールド左側の **📎 クリップアイコン** をクリック
2. 送信したいファイルを選択（最大 10MB）
3. ファイルが自動的にアップロードされ、メッセージとして投稿されます

### ファイルのダウンロード

メッセージに表示された **「Attachment」** リンクをクリックするとファイルを開けます。

対応ファイル形式に制限はありませんが、画像（JPEG / PNG / GIF）はインラインプレビューが表示されます。

---

## 9. メッセージ検索

全チャンネルのメッセージを横断して検索できます。

### 検索の使い方

1. サイドバー上部の **🔍 検索** ボタンをクリック
2. 検索キーワードを入力（入力後 300ms でリアルタイム検索）
3. 検索結果からメッセージをクリックすると、対象チャンネルに移動します

### 検索の仕様

- MongoDB のテキストインデックスを使用した全文検索
- 対象: `messages.content` フィールド
- スペース区切りで複数キーワードの AND 検索が可能

## 10. コーディングボット

Dockerfile をアップロードしてコーディングボットを登録できます。

> **現状**: ボット登録・バリデーション・一覧取得のみ実装済み。  
> コンテナビルド・実行・Copilot SDK との統合は今後実装予定です。

### ボットの登録

API 経由で操作します:

```bash
curl -X POST http://localhost:8000/api/bots \
  -H "Authorization: Bearer {token}" \
  -F "name=my-bot" \
  -F "description=My coding bot" \
  -F "container_file=@./Dockerfile"
```

### Dockerfile のバリデーション（登録なし）

```bash
curl -X POST http://localhost:8000/api/bots/validate \
  -H "Authorization: Bearer {token}" \
  -F "container_file=@./Dockerfile"
```

### ボット一覧の取得

```bash
curl http://localhost:8000/api/bots \
  -H "Authorization: Bearer {token}"
```

### ボットのステータス

| ステータス | 説明 |
|-----------|------|
| `registered` | 登録済み（初期状態） |
| `building` | コンテナビルド中 |
| `ready` | 実行可能 |
| `error` | エラー発生 |
