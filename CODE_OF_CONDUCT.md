# 共同作業者行動規範

## 私たちのコミットメント

協力的な環境を維持するため、私たちはメンバーとして、私たちのプロジェクトへの貢献や議論において、嫌がらせのない経験を提供することを約束します。

## 私たちの基準

好ましい行動の例：

* 丁寧な振る舞い
* 他人の異なる見解や経験を尊重すること
* 建設的なフィローバックを受け入れること
* 自分の過ちを認め、そこから学ぶこと
* コミュニティ全体にとって最善のことに集中すること

受け入れられない行動の例：

* 性的な言葉や画像の使用、不適切な性的関心や進展
* 嫌がらせ、侮辱的または軽蔑的なコメント、個人的または政治的な攻撃
* 公的または私的な嫌がらせ
* 許可なく他人の住所やメールアドレスなどの個人情報を公開すること
* 仕事環境において不適切と考えられるその他の行為

## 責任

プロジェクトのメンテナーは、許容される行動の基準を明確にし、不適切な行動があった場合には適切な法的措置を取る責任があります。

## 適用範囲

この行動規範は、プロジェクトが代表されるかどうかにかかわらず、プロジェクト内および公的な場において適用されます。

## 執行

嫌がらせやその他の不適切な行動は、プロジェクトのメンテナー（[security.md](SECURITY.md) に記載の連絡先）に報告してください。すべての苦情は迅速かつ公正に調査されます。
